Blinkit Sales Performance Dashboard
Project Overview

This project presents an interactive Sales Performance Dashboard for Blinkit, built using Power BI. The objective of this project is to analyze grocery sales data and uncover meaningful insights related to product performance, outlet performance, and customer preferences.

The dashboard helps stakeholders understand key business metrics such as total sales, product demand, outlet performance, and customer ratings. It enables data-driven decision-making by providing a clear visual representation of sales patterns across different product categories and outlet types.

Business Objective

The main objective of this project is to analyze Blinkit's grocery sales data to:

Evaluate overall sales performance

Identify top-performing product categories

Analyze the impact of outlet characteristics on sales

Understand customer preferences through product ratings

Provide insights that can help improve business strategies

Dataset Description

The dataset used in this project contains information about Blinkit grocery items and outlet characteristics.

Columns Included

Item Fat Content – Indicates whether the product is low fat or regular

Item Identifier – Unique identifier for each product

Item Type – Category of the product (e.g., dairy, snacks, fruits)

Outlet Establishment Year – Year in which the outlet was established

Outlet Identifier – Unique ID for each outlet

Outlet Location Type – Tier level of the outlet location (Tier 1, Tier 2, Tier 3)

Outlet Size – Size of the outlet (Small, Medium, Large)

Outlet Type – Type of outlet (Supermarket, Grocery Store, etc.)

Item Visibility – Visibility percentage of the item in the store

Item Weight – Weight of the product

Sales – Sales generated by the product

Rating – Customer rating for the product

Data Cleaning and Preparation

Data preprocessing was performed in Power Query to ensure accurate analysis.

Key steps included:

Removing duplicate records

Handling missing values

Correcting data types

Standardizing item fat content categories

Preparing the dataset for dashboard visualization

Key Performance Indicators (KPIs)

The dashboard highlights several important business metrics:

Total Sales – Overall revenue generated from all items

Average Sales – Average sales value per product

Average Rating – Average customer rating across products

Total Items – Number of unique products in the dataset

Total Outlets – Number of outlets contributing to sales

Dashboard Insights

The dashboard provides insights through multiple visualizations such as:

Sales by Item Type

Sales by Outlet Location Type

Sales by Outlet Size

Sales by Outlet Type

Sales by Item Fat Content

Item Visibility vs Sales relationship

Average Rating by Product Category

These visualizations help identify trends, high-performing categories, and potential business opportunities.

Conclusion

The Blinkit Sales Dashboard provides a comprehensive overview of grocery sales performance and enables stakeholders to explore sales trends across different products and outlets. The interactive nature of the dashboard allows users to filter and analyze data efficiently, supporting better business decision-making.
